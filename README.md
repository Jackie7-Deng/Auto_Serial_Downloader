# Auto Serial Downloader (Serial Downloader v1.0)

A minimal Windows host tool (WinForms/.NET) for Infineon/Cypress USB-Serial Bridge devices to:
- **Connect / Disconnect**: detect target device by VID/PID using `cyusbserial.dll` APIs.
- **Download**: program a pre-generated `*.cyusb` configuration file by invoking `fwDownload.exe`.

This project is intended as a reference framework for customers who want to integrate USB-Serial configuration programming into their own production/test applications.

---

## UI Overview
- VID / PID / SN input
- Browse `*.cyusb` file
- Connect/Disconnect button (locks VID/PID when connected)
- Download button (runs `fwDownload.exe`)
- Optional progress overlay during programming

---

## Prerequisites (Runtime)
- Windows x64
- USB-Serial device manufacturing interface enabled
- Device bound to **CYUSB3** driver (CyUSB3.sys) for enumeration/open using the SDK API

---

## Repository Contents
- `assets/CY.ico`  
  App icon.
- `configs/default_*.cyusb`  
  Example configuration file (generated by USB-Serial Configuration Utility).
- `sdk/cyusbserial.dll`  
  Cypress USB-Serial SDK runtime library (x64).
- `tools/fwDownload.exe`  
  Cypress/Infineon command line configuration tool used to program `*.cyusb`.

> Note: `cyusbserial.dll` and `fwDownload.exe` are redistributed here for demo/reference usage.
> Please follow Infineon/Cypress licensing rules if you redistribute this project externally.

From the project folder:
```powershell

```

bin\Release\net8.0-windows\win-x64\publish\
Distribute the entire publish folder (zip) to end users.

## References

1. USB-Serial SDK documentation and header file:
   C:\Program Files (x86)\Cypress\USB-Serial SDK\library\cyusbserial\CyUSBSerial.h
2. Command line tool usage (fwDownload.exe):
   fwDownload.exe -c <config_file.cyusb> -vid <vid> -pid <pid> -sn <serial_number>
   (See Infineon/Cypress KBA: "USB-Serial command line configuration utility")
3. Windows driver note:
   CyGetListofDevices enumerates devices bound to the CYUSB3 driver on Windows.
